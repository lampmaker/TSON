## What is TSON?

TSON (Typed Structured Object Notation) is a human-friendly superset of JSON that supports:
- 📊 **Tables and structured data**
- 🎨 **Flexible formatting options** 
- 🔧 **Configurable parsing and creation**
- 📝 **Human-readable syntax**
